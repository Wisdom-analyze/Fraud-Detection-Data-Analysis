# Fraud-Detection-Power-BI
Fraud detection dashboard using MySQL, Ms Excel and Power BI
## Project Overview 
This project analzes finacial transaction data to identify patterns associated with fraudulent activites.
The goal is to uncover insights that can halp financial institutions detect suspicious transactions and reduce fraud risks.
## Tools Used 
Excel- Data cleaning
MySQL- Data storage and Querying
Power BI- Data visualization
GitHub- Project Documentation
